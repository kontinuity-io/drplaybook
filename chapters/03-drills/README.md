# Chapter 03: Running DR Drills

**Through-line:** A DR plan that has never been executed is not a DR plan — it's a hypothesis. This chapter teaches you to design, execute, and record DR drills that produce credible evidence of actual capability.

**What you build:** A drill plan, an execution runbook, and a post-drill evidence package.

**Prerequisites:** Chapters 00 and 01 complete. At least one Recovery Group defined with replication configured.
