# Chapter 07: Incident Response

**Through-line:** A DR event is not a scheduled drill. This chapter prepares you for the real thing: the declaration decision, the first 30 minutes, the failover execution, and the path back to primary.

**What you build:** An incident response runbook and a failback checklist for your environment.

**Prerequisites:** Chapters 00–03 complete. At least one successful DR drill completed.
