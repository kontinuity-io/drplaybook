# Chapter 08: DR Tooling

**Through-line:** Manual DR is the enemy of reliable DR. This chapter maps the Kontinuity OSS toolkit — nine open-source CLI tools that automate the most common DR programme tasks from posture scanning to runbook generation.

**What you build:** A configured toolkit tailored to your environment, with each tool integrated into your Recovery Group monitoring and drill workflow.

**Prerequisites:** Chapter 00 complete. One or more completed chapters from 01–07 recommended.

**About this chapter:** All tools are published under Apache 2.0 by Kontinuity Labs. Source: github.com/kontinuity-io. The Kontinuity platform (available at getkontinuity.com) provides the orchestration layer above these tools — automating Recovery Group monitoring, drill execution, and compliance evidence generation.
